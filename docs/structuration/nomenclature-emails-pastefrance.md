## 5. Cellules

Format : `cellule.<code>.<nomcellule>@pasteffrance.fr`

Le `<code>` est un code court à 3 lettres identifiant la sous-section de rattachement (cf. table ci-dessous), utilisé pour éviter toute collision entre cellules de sous-sections différentes portant un nom similaire.

Exemples :
- `cellule.par.republique@pasteffrance.fr`
- `cellule.pac.nicecentre@pasteffrance.fr`
- `cellule.naq.bordeauxsud@pasteffrance.fr`
